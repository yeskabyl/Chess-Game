# ♔ Chess Game

Полнофункциональная интерактивная игра в шахматы на чистом HTML, CSS и JavaScript (ES6+). Классическая шахматная доска с поддержкой всех стандартных правил игры, включая проверку на шах, мат и пат.

## 🎮 Возможности

### ✨ Основной функционал
- **Полная поддержка шахматных правил** для всех типов фигур
- **Валидация ходов** для каждой фигуры (пешка, конь, слон, ладья, ферзь, король)
- **Система таймеров** для каждого игрока (10 минут по умолчанию)
- **Поочередные ходы** (белые ходят первыми)
- **История всех ходов** в стандартной нотации

### 🔍 Продвинутые функции
- **Проверка на шах** — красное выделение король под угрозой
- **Определение мата** — автоматическое завершение игры
- **Определение пата** — ничья при отсутствии валидных ходов
- **Защита от невалидных ходов** — король не может остаться под шахом
- **Автоматическое превращение пешки** в ферзя при достижении последнего ряда

### 🎨 Пользовательский интерфейс
- **Красивая шахматная доска** 8×8 с классическими цветами
- **Unicode символы фигур** (♔♕♖♗♘♙)
- **Подсветка выбранной фигуры** синим цветом
- **Визуализация доступных ходов** зеленым полупрозрачным выделением
- **Последний ход** отмечен едва заметным синим выделением
- **Темный минималистичный интерфейс** с современным дизайном
- **Панели игроков** с аватарами и таймерами (плавная подсветка активного игрока)
- **Боковая панель** с историей ходов, статусом игры и срубленными фигурами
- **Адаптивный дизайн** для всех устройств (ПК, планшет, мобильный)

### 🎯 Управление игрой
- **Новая игра** — начать игру с нуля
- **Отменить ход** — откатиться на один ход назад
- **Сдаться** — завершить игру поражением
- **Предложить ничью** — кнопка для предложения ничьей

## 📋 Правила игры

### Движение фигур
- **Пешка (♙♟)**: Движется вперед на 1 клетку (или на 2 с начальной позиции), захватывает по диагонали, превращается в ферзя на последней линии
- **Конь (♘♞)**: L-образные ходы (2+1 клеток)
- **Слон (♗♝)**: Диагональные ходы на любое расстояние
- **Ладья (♖♜)**: Горизонтальные и вертикальные ходы на любое расстояние
- **Ферзь (♕♛)**: Комбинация слона и ладьи
- **Король (♔♚)**: Ход на одну клетку в любом направлении

### Особые условия
- **Шах** — король под угрозой захвата
- **Мат** — нет валидных ходов и король под шахом (игра окончена)
- **Пат** — нет валидных ходов, король не под шахом (ничья)
- **Защита от невалидных ходов** — нельзя двигать фигуру, если король останется под шахом

## 🎮 Как играть

1. **Выберите фигуру** — кликните на фигуру вашего цвета
2. **Доступные ходы** — появятся зеленые точки с возможными ходами
3. **Сделайте ход** — кликните на одно из доступных полей
4. **Смена игрока** — ход автоматически переходит к противнику
5. **Следите за таймером** — активный игрок подсвечивается зеленым
6. **История ходов** — все ходы сохраняются в боковой панели

## 🏗️ Архитектура

### Структура файла
```
chess.html          # Один файл со всем кодом
├── <style>         # CSS стили (интерфейс + доска)
├── <script>        # JavaScript (игровая логика)
└── <body>          # HTML структура
```

### Основные классы JavaScript
- **ChessGame** — главный класс игры
  - `createInitialBoard()` — инициализация доски
  - `getValidMoves()` — получение валидных ходов для фигуры
  - `makeMove()` — выполнение хода
  - `checkGameStatus()` — проверка на шах/мат/пат
  - `isSquareUnderAttack()` — проверка, находится ли поле под атакой

## 💡 Особенности реализации

### Валидация ходов
- Каждый тип фигуры имеет свою функцию для вычисления возможных ходов

## 🎨 Кастомизация

Вы можете легко изменить:
- **Цвета доски** — обновите `--board-light` и `--board-dark` в CSS
- **Таймер** — измените значение в `this.whiteTime = 600` (в секундах)
- **Шрифт** — замените `'Inter'` на любой другой в `font-family`
- **Размер доски** — обновите `grid-template-columns: repeat(8, 60px)`

## 🐛 Известные ограничения

- Рокировка (castling) — не реализована
- Взятие на проходе (en passant) — не реализована
- 3-кратное повторение позиции — не отслеживается
- 50-ходовое правило — не отслеживается

Эти правила могут быть добавлены в будущих версиях.
